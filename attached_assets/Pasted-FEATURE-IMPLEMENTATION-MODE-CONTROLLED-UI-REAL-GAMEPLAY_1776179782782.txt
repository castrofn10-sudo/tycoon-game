B) CONECTAR PESQUISAS A DESBLOQUEIOS REAIS
Toda pesquisa concluída deve poder liberar funcionalidades reais em sistemas do jogo, principalmente:
- construção de consoles
- criação de jogos
- recursos online
- melhorias tecnológicas
- negócios/distribuição quando aplicável

REGRA CENTRAL:
Pesquisas relacionadas a consoles devem liberar opções reais no menu de construção de consoles.

EXEMPLOS:
- CPU / arquitetura / GPU / mídia / bateria / cooling / áudio / portabilidade:
  devem aparecer como opções no construtor de consoles quando pesquisadas
- pesquisas de online:
  devem liberar recursos online reais no desenvolvimento de jogos/plataformas quando aplicável
- pesquisas de engines/gameplay:
  devem liberar recursos jogáveis coerentes para criação de jogos
- pesquisas de negócio:
  devem liberar opções de mercado, distribuição, contratos, loja digital, etc., se já houver sistema correspondente

C) LIBERAÇÃO POR ÉPOCA

- antes do ano exigido: pesquisa bloqueada
- ao alcançar o ano exigido: pesquisa fica disponível
- após concluir: desbloqueio real é concedido

A disponibilidade deve respeitar:
- ano mínimo da pesquisa
- pré-requisitos da árvore, se existirem
- estado salvo atual do jogador
